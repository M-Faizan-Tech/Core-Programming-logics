# OOP Object Lifecycle (Constructors & Destructors)

This project explores the compilation lifecycle of class entities, automatically initializing and tearing down object boundaries.

## 🧠 Logic & Lifecycle Phases
- **Default Initialization:** Sets standard structural default values when an object is declared without parameters.
- **Parameterized Allocation:** Dynamically maps custom runtime properties straight into private data containers upon creation.
- **Automatic Destruction (`~`):** Operates on stack frame pop; when `main()` reaches execution termination, the system automatically runs the destructor to free active system scopes.

## 🛠️ Concepts Covered
- Default & Parameterized Constructors
- Object Lifecycle & Scope Management
- Destructor Cleanups (`~`)
