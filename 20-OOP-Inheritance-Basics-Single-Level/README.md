# OOP Single-Level Inheritance Foundations

This project establishes the structural flow of class hierarchy where a child class extends a single base parent class.

## 🧠 Logic & Code Reusability
- **Protected Access Specifier:** The fields `name` and `age` inside the parent class are marked as `protected`. This keeps them completely locked away from external files but fully readable by the child class.
- **Single-Level Flow:** The `Employee` class derives directly from `Person`, instantly reusing its attributes without duplicating code footprints.

## 🛠️ Concepts Covered
- Base vs Derived Classes
- Single-Level Object Hierarchies
- Protected Variable Visibility Bounds
