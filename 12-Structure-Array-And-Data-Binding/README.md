# Structure Arrays and Database Binding

An implementation showcasing custom user-defined data structures combined with array bundling to mimic basic local flat-file databases.

## 🧠 Logic & Architecture
- **Data Encapsulation:** Uses a custom `struct Book` to bind diverse variable types (`int`, `string`, `float`) into a single logical unit.
- **Array of Structures:** Declares `library[100]` to hold multiple book profiles sequentially in memory. Loop indexing is utilized to seamlessly traverse, read, and output clean formatted records.

## 🛠️ Concepts Covered
- Struct Complex Layouts
- Complex Array Data Structuring
- Input Stream Flush Management (`cin.ignore()`)
