# Two-Dimensional Matrix Multiplication

An advanced algorithmic script that tracks multi-dimensional structures and applies standard row-to-column matrix multiplication math.

## 🧠 Logic & Algorithm
- Requires 3 nested loops to execute matrix multiplication:
  1. **Outer loop `i`:** Manages rows of Matrix A.
  2. **Middle loop `j`:** Manages columns of Matrix B.
  3. **Inner loop `k`:** Iterates through common dimensions to execute the dot-product mathematical expression: `mult[i][j] += a[i][k] * b[k][j]`.
