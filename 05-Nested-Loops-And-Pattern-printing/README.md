# Nested Loops and Pattern Printing Formations

A logic-building script dedicated to mastering loop control boundaries through 2D pattern coordinate printing.

## 🧠 Logic & Algorithm
- **Outer Loop:** Manages the vertical grid rows (`i`).
- **Inner Loop:** Controls the horizontal grid elements (`j`) printed on each specific line.
- **Inverted Scaling:** Uses a decrementing outer loop (`i--`) to progressively shrink the horizontal count, rendering an inverted number layout.

## 🛠️ Concepts Covered
- Nested Loop Coordination
- Coordinate Geometry Mapping (`i` rows, `j` columns)
- Decrementing Loop Boundaries
