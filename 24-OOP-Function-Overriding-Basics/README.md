# OOP Function Overriding Basics

This module demonstrates how a derived class can override and provide a specialized implementation of a method that is already defined in its base class.

## 🧠 Logic & Method Redefinition
- **Method Shadowing:** When a function in the child class has the exact same name, return type, and parameters as a function in the parent class, the child's method shadows (overrides) the parent's version.
- **Static Resolution:** By default, without specialized keywords, object instances resolve these functions based on their reference type at compile time.

- Function Overriding Signatures
