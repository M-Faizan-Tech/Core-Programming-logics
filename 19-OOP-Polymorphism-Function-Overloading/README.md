# Compile-Time Polymorphism (Function Overloading)

An architectural presentation showcasing functional polymorphism by compiling multiple signatures under a single identifier.

## 🧠 Logic & Binding Methods
- **Static Binding:** The C++ compiler analyzes function arguments at compile time (type, number, and order) rather than at runtime.
- **Signature Customization:** Three distinct mathematical implementations share the identical `computeArea` name but are completely isolated by changing parameter definitions (`int`, `int, int`, and `double`).

## 🛠️ Concepts Covered
- Compile-Time Polymorphism
- Static Scope Binding
- Type-Driven Method Disambiguation
