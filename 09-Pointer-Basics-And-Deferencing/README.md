# Pointer Basics and Memory Dereferencing

A fundamental memory-mapping utility that isolates hexadecimal RAM addresses from underlying integer values using C++ pointers.

## 🧠 Logic & Memory Mapping
- **Address-Of Operator (`&`):** Extracts the unique hardware memory location where a variable is stored.
- **Pointer Variables (`*ptr`):** A specialized reference variable designed exclusively to house memory addresses.
- **Dereferencing Operator (`*`):** Pierces through the pointer allocation to directly modify or extract the real value inside the target memory address.

## 🛠️ Concepts Covered
- Memory Address Extraction (`&`)
- Pointer Data Types (`int*`)
- Value Dereferencing Manipulation
