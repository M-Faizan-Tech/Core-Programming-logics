# OOP Multi-Level Inheritance Flow

An algorithmic blueprint map demonstrating deep linear inheritance chaining where a child class acquires structures across multiple generational levels.

## 🧠 Logic & Inheritance Levels
- **Linear Prototyping:** Class `Device` sets the baseline, `Computer` inherits from `Device`, and finally `Laptop` inherits from `Computer`.
- **Chained Access:** The terminal leaf node class (`Laptop`) effortlessly accesses private/protected attributes residing two levels above in the grandparent block.

## 🛠️ Concepts Covered
- Linear Generational Hierarchies
- Deep Variable Propagation
- Multi-Level Scope Resolution
