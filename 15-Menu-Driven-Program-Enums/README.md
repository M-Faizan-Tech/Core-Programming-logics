# Menu-Driven Application via Enumerations

An interactive terminal application focused on creating professional CLI menus utilizing structured control switch flags and Enums.

## 🧠 Logic & Flow Structure
- **User Enumeration (`enum`):** Maps integer user inputs directly to human-readable states (`ADD = 1`, `SUBTRACT`, etc.) to enhance code clean-coding practices.
- **Infinite Evaluation Loop (`while(true)`):** Persists runtime interaction until the specific `EXIT` signal breaks the execution thread.
- **Defensive Input Handling:** Filters zero-division attempts inside the math compiler blocks to prevent core terminal crashes.

## 🛠️ Concepts Covered
- Explicit Enumerations (`enum`)
- Persistent CLI Loops & Break Conditions
- Switch-Case Control Branching
