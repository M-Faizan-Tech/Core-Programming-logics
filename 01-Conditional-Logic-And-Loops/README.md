# Conditional Logic and Loop Analysis

A unified core program to analyze integer boundaries, detect prime number status, and calculate factorial values.

## 🧠 Logic & Algorithm
1. **Input Guard:** Validates if the user input is positive (`num > 0`).
2. **Prime Logic:** Runs a loop from `2` up to `num/2`. If any number cleanly divides it (`num % i == 0`), the flag flips to false.
3. **Factorial Logic:** Iterates a counter from `1` to `num`, continuously multiplying the values into an accumulator.

## 🛠️ Concepts Covered
- Conditional Branches (`if-else`)
- Loop Controls (`for` loop, `break` statement)
- Boolean Flags and Type Validation
