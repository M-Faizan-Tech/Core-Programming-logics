# Project 2: Array Searching, Min/Max Extraction, and Bubble Sorting

This project processes numerical arrays to extract dynamic extremes and sort data chronologically.

## 🧠 Logic & Algorithm
- **Extreme Search:** Initializes baseline variables with array index `0`, then compares each subsequent node to find absolute minimum and maximum values.
- **Bubble Sort:** Uses nested iterations to compare adjacent items (`arr[j]` vs `arr[j+1]`). If the left item is greater, they swap memory slots using a temporary storage container.
