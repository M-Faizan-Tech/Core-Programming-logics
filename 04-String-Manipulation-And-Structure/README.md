# String Manipulation and User-Defined Structures

A dual-purpose utility that demonstrates memory indexing through string reversal algorithms and structured data bundling using C++ structures.

## 🧠 Logic & Algorithm
- **String Reversal:** Reads the length of the string, then sets up a loop that counts *backward* from `length - 1` down to `0`. It extracts each character sequentially and appends it to a new string container.
- **Structures (`struct`):** Creates a custom data type to encapsulate distinct variable forms (`string`, `int`, `float`) under a unified entity, simulating basic database entries.

## 🛠️ Concepts Covered
- Backward Loop Indexing
- Input Buffer Clearing (`cin.ignore()`)
- Custom Data Structures (`struct`)
